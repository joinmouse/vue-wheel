    # Grid 栅格

    24 栅格系统

    # 概述

    布局的栅格化系统，我们是基于行（row）和列（col）来定义信息区块的外部框架，以保证页面的每个区域能够稳健地排布起来。

    • 通过row在水平方向建立一组col
    • 内容应当放置于col内
    • 栅格系统中的列是指 1 到 24 的值来表示其跨越的范围。例如，四个等宽的列可以使用 来创建

</WhRow>
